# Tasks Completed

## ✅ Task 1 – Cloud File Storage System

### Description

A Flask-based cloud file storage application that allows users to upload, download, view, and delete files securely.

### Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite
- SQLAlchemy

### Features

- Upload Files
- Download Files
- View Uploaded Files
- Delete Files
- Dashboard
- Local Storage
- AWS S3 Integration (Coming Soon)

## ✅ Task 2 – Cloud Data Deduplication System

### Technologies Used

- Python
- Flask
- Pandas
- SQLite
- SQLAlchemy
- HTML
- CSS

### Features

- CSV Upload
- Duplicate Detection
- Data Validation
- Dashboard
- Search Records
- Delete Records

### Technologies Used

- Flask-SQLAlchemy

- HTML
- CSS

### Features

- Course Management

- Responsive User Interface
