# CampusCloud Student Management System

## Project Overview

CampusCloud Student Management System is a web-based application developed using Python and Flask. It helps manage student records, courses, attendance, and grades through an easy-to-use interface.

---

## Features

- Dashboard
- Student Management (Add, Edit, Delete, View)
- Course Management
- Attendance Management
- Grade Management
- SQLite Database
- Responsive User Interface
- Flask Web Framework

---

## Technologies Used

- Python
- Flask
- Flask-WTF
- Flask-SQLAlchemy
- SQLite
- HTML
- CSS

## Installation

Create a virtual environment

```
python -m venv venv
```

Activate virtual environment

Windows

```
venv\Scripts\activate
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
python app.py
```

Open

```
http://127.0.0.1:5000
```
